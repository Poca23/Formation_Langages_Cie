A.Installation de MySQL

1. Installation de MySQL avec le terminal

  • Sur Linux (Ubuntu) : Si MySQL n'est pas encore installé, vous pouvez l'installer en exécutant les commandes suivantes dans votre terminal : 
                                
bash
sudo apt update
sudo apt install mysql-server
  • Sur Windows ou macOS : Si vous êtes sur un autre système, vous pouvez télécharger MySQL à partir de ce lien officiel. Il suffit de suivre les instructions pour installer MySQL sur votre machine.
  • Connexion à MySQL depuis VS Code

2. Vérification du fonctionnement de MySQL avec le terminal

  • Une fois MySQL installé, vous pouvez démarrer le serveur MySQL avec : 
                                
bash
sudo systemctl start mysql
  • Pour vérifier que MySQL fonctionne correctement : 
                                
sudo systemctl status mysql
    Vous devriez voir une confirmation que MySQL est en cours d'exécution.

3. Connexion à MySQL avec le terminal

  • Pour se connecter à MySQL, vous pouvez utiliser la commande suivante dans le terminal : 
                                
bash
sudo mysql -u root -p
    Il vous sera demandé de saisir le mot de passe pour l'utilisateur root que vous avez configuré lors de l'installation de MySQL.

B.Installation de l'extension MySQL pour VS Code

4. Installation de l'extension MySQL pour VS Code

  • Ouvrez Visual Studio Code.
  • Allez dans l'onglet des Extensions (icône carrée dans la barre latérale à gauche).
  • Recherchez "MySQL" et installez l'extension "MySQL" de Jun Han (ou une extension équivalente que vous préférez).
  • Une fois l'extension installée, vous pourrez vous connecter à votre base de données directement depuis VS Code pour exécuter des requêtes SQL.

C.Connexion à MySQL depuis VS Code

1. Installer l'extension MySQL dans VS Code

  • Ouvrez VS Code.
  • Cliquez sur l’icône des extensions (à gauche, représentée par quatre blocs empilés).
  • Dans la barre de recherche, tapez "MySQL".
  • Installez l’extension appelée MySQL by Weijan Chen (ou une autre populaire et bien notée).

2. Configurer la connexion à MySQL

  • Une fois l’extension installée, ouvrez le panneau latéral en cliquant sur l’icône associée à l’extension (souvent un logo MySQL dans la barre latérale).
  • Cliquez sur Add Connection ou un bouton similaire.
  • Remplissez les informations suivantes :
    • Host : localhost
    • Port : 3306 (par défaut, sinon ajustez si différent).
    • User : root (ou un autre utilisateur que vous avez créé).
    • Password : Entrez votre mot de passe utilisateur MySQL.

3. Testez de connexion en exécutant des requêtes SQL

  1. Développez une base de données pour voir ses tables et colonnes.
    • Si la connexion réussit, vous verrez votre connexion listée dans le panneau latéral MySQL.
    • Cliquez sur la connexion pour l’ouvrir. Vous devriez voir toutes les bases de données (comme blog_cinema).
  2. Vérification de la connexion
    • Ouvrir l'éditeur de requêtes SQL
      Vous avez mentionné avoir cliqué sur "Open Query". Cela ouvre un éditeur où vous pouvez écrire et exécuter vos requêtes SQL.
    • Écrire une première requête
      Une fois l'éditeur ouvert, tapez une requête simple pour vérifier que la connexion fonctionne bien et que vous pouvez interagir avec MySQL. Essayez la commande suivante : 
                                                  
sql
SHOW DATABASES;
      Cette commande listera toutes les bases de données disponibles sur votre serveur MySQL. Si la connexion est réussie, vous devriez voir une liste de bases de données, y compris celle que vous avez créée (par exemple, blog_cinema).
  3. Exécuter la requête
    Pour exécuter la requête, placez vous après le ' ; ' de la requête et faite : Ctrl + Enter.
  4. Accéder à une base de données
    Pour sélectionner une base de données spécifique et commencer à travailler dessus, utilisez la commande suivante : 
                                        
sql
USE blog_cinema;
    Ce qui donnera : 
                                        
USE blog_cinema

AffectedRows : 0
    Cela vous permettra de travailler sur la base de données blog_cinema.
  5. Vérifier les tables de la base de données
    Une fois que vous avez sélectionné votre base de données, vous pouvez afficher les tables qu'elle contient avec cette commande : 
                                        
sql
SHOW TABLES;

D.Suppression / Modifications / manipulation de mysql avec le terminal

5. Suppression d'une base de donnée sur MySQL

  • On peut utiliser : 
                                
bash
sql
DROP DATABASE nom-de-la-bdd;
    Cette commande supprimera définitivement la base de données, alors il faut s'assurer de vouloir la supprimer.
  • Vous pouvez également vérifier que la base a été supprimée après l'avoir exécutée avec : 
                                
bash
sql
SHOW DATABASES;
  • !!! ATTENTION, c'est irréversible !!!

6. Sortir de MySQL dans le terminal

  • Taper la commande : 
                                
bash
sql
EXIT;
    ou : 
                                
bash
sql
QUIT;

7. Pas de connexion à MySQL avec les bons privilèges

  1. Se connecter en tant que root (utilisateur administrateur de MySQL).
    Lorsque vous utilisez MySQL, le compte root est souvent utilisé pour l'administration. Tentez de vous connecter avec le compte root en spécifiant un mot de passe : 
                                
bash
sudo mysql -u root -p
    Entrez le mot de passe de l'utilisateur root que vous avez configuré lors de l'installation de MySQL. (password) 
                                
bash
sql
QUIT;
  2. Essayer à nouveau
    Une fois connecté, vous pouvez vérifier que MySQL fonctionne correctement en exécutant la commande suivante : 
                                
SHOW DATABASES;

8. Oubli de spécification d'un mot de passe

  • Changer le mot de passe de l'utilisateur (si nécessaire)
    Si vous avez des difficultés à vous connecter avec root ou si vous souhaitez configurer un autre utilisateur, vous pouvez aussi réinitialiser les mots de passe ou attribuer de nouveaux privilèges.
    Réinitialiser le mot de passe root (si nécessaire) : 
                                
bash
sudo mysql
    Ensuite, dans l'invite MySQL, exécutez la commande suivante pour modifier le mot de passe root : 
                                
ALTER USER 'root'@'localhost' IDENTIFIED BY 'new_password';
FLUSH PRIVILEGES;
    Remplacez new_password par le mot de passe souhaité.

9. Création d'un nouvel utilisateur (optionnel)

  • Création d'un nouvel utilisateur (optionnel)
    Si vous voulez créer un autre utilisateur avec des privilèges, vous pouvez le faire une fois connecté à MySQL : 
                                
bash
sql
CREATE USER 'greta'@'localhost' IDENTIFIED BY 'your_password';
GRANT ALL PRIVILEGES ON *.* TO 'greta'@'localhost' WITH GRANT OPTION;
FLUSH PRIVILEGES;
    Cela crée un nouvel utilisateur greta avec tous les privilèges.
