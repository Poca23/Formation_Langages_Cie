1. Modèle Conceptuel de Données (MCD)
Avant de commencer à créer des tables, il est important de réfléchir à la structure de la base de données. D'après les fonctionnalités que tu as dans ton projet, voici les principales entités que tu pourrais avoir :

1-Utilisateurs : Pour gérer les comptes utilisateurs.
2-Films : Pour stocker les informations sur les films.
3-Catégories : Pour organiser les films par catégories.
4-Favoris : Liste des films favoris des utilisateurs.
5-Visionnés : Liste des films visionnés par un utilisateur.


Entités possibles :

1/Utilisateurs :
id (clé primaire)
nom
prenom
email
mot_de_passe (hashé)
date_inscription
etc.

2/Films :
id (clé primaire)
titre
synopsis
date_sortie
image
notation
critique
trailer


3/Catégories :
id (clé primaire)
nom


4/Favoris :
id (clé primaire)
utilisateur_id (clé étrangère)
film_id (clé étrangère)


5/Visionnés :
id (clé primaire)
utilisateur_id (clé étrangère)
film_id (clé étrangère)
date_visionnage
