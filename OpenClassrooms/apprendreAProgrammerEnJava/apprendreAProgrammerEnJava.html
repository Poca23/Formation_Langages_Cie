Apprendre à programmer en Java

Imaginez que vous êtes un horloger, assemblant avec précision chaque rouage, chaque engrenage, pour créer une montre complexe et élégante.

Et si vous pouviez construire cette montre en remplaçant chaque élément par des lignes de code ?

C’est ce que Java va vous permettre de faire et c’est ce que vous allez découvrir dans ce cours !

Vous commencerez par installer votre environnement de développement avant de prendre en main les concepts de base de Java.

Vous apprendrez ensuite à écrire votre premier programme en déclarant des variables, créant des fonctions et mettant en place des boucles.

Ensuite, vous découvrirez la programmation orientée objet en écrivant vos premières classes.

Enfin, vous approfondirez votre maîtrise de Java par l’utilisation du lambda, l’application de traces dans votre code ou encore la manipulation des flux.

Prêt à commencer votre aventure Java ?

Suivez notre cours pour devenir un développeur Java compétent et créatif.


Objectifs pédagogiques

À la fin de ce cours, vous serez capable de :


  • maîtriser les bases de Java ;
  • prendre en main la programmation orientée objet ;
  • approfondir votre maîtrise de Java.

Pour suivre ce cours, vous devez maîtriser les compétences suivantes :


  • Connaître les bases du terminal
  • Manipuler des fichiers avec un terminal
  • Décrire le fonctionnement d'un algorithme
  • Choisir le type et la structure de données adéquats
  • Appliquer les concepts avancés de l’algorithmique

Vous pouvez suivre ce cours pour développer vos compétences :


  • Apprenez à utiliser la ligne de commande dans un terminal
  • Découvrez le fonctionnement des algorithmes

Sommaire :

  • Partie 1 - Maîtrisez les bases de Java
  • Partie 2 - Prenez en main la programmation orientée objet
  • Partie 3 - Approfondissez votre maîtrise de Java
