System.out.println("Programme d'entrainement aux nouveaux apprentis");

do {

Vous décidez de la débloquer et lui indiquer la raison :


  • La variable valeur est typée par le type primitif double. Une instruction while ne peut pas fonctionner avec une comparaison sur un double.
  • La boucle do { } while() n’est pas adaptée, il aurait fallu utiliser une boucle while afin d’évaluer d’abord la valeur puis exécuter l’instruction.
  • La variable valeur est déclarée au sein du bloc do { } et de ce fait l’instruction while n’y accède pas. Il faut déclarer la variable en dehors de la boucle.
