Apprendre à Programmer en Javascript

Dans ce cours, vous découvrirez les bases du langage de programmation JavaScript :

sa syntaxe, sa logique de programmation… mais pas que !
Vous serez également guidé, pas à pas, dans la réalisation d’un site simple, fonctionnel et dynamique.

Alors, prêt à apprendre à programmer en JavaScript ?
C’est parti !

Objectifs pédagogiques

À la fin de ce cours, vous serez capable de :


  • manipuler des données avec JavaScript ;
  • écrire un fichier JavaScript ;
  • faire interagir JavaScript avec votre page web ;
  • créer un formulaire de saisie de données.

Prérequis :

Pour suivre ce cours, vous devez maîtriser les bases de la programmation en HTML5 et CSS3. Vous pouvez développer ces compétences en suivant le cours Créez votre site web avec HTML5 et CSS3.
Outils nécessaires :
Dans le cadre de ce cours, vous allez réaliser un site web.

Vous aurez besoin :


  • d’un navigateur web à jour (Chrome, Safari, Firefox ou encore Edge) ;
  • et d’un éditeur de texte (Visual Studio Code) que vous allez télécharger.
  • Vous utiliserez aussi GitHub pour consulter le code du site web et les corrigés des exercices.

Sommaire :

  • Partie 1 - Manipulez des données avec JavaScript
  • Partie 2 - Écrivez votre premier fichier JavaScript
  • Partie 3 - Faites interagir JavaScript avec une page web
  • Partie 4 - Créez un formulaire de saisie de données

Le but de ce cours n’est pas d’être exhaustif, mais de vous donner les bases nécessaires pour comprendre la logique de programmation.
