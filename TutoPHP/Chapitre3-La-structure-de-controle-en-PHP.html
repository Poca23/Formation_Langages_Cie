Chapitre 3 - Les structures de contrôle en PHP

Les conditions : if, else, elseif

Les structures conditionnelles permettent d'exécuter des blocs de code en fonction de certaines conditions. En PHP, on utilise principalement les structures suivantes :

  • if : Vérifie une condition. Si elle est vraie, le code à l'intérieur du bloc est exécuté.
  • else : Exécute le code à l'intérieur du bloc si la condition du if est fausse.
  • elseif : Permet d'ajouter d'autres conditions à tester si le premier if échoue.

Exercice :

  • Créez un script qui vérifie si un nombre est positif, négatif ou égal à zéro. Utilisez if, else et elseif pour afficher les messages appropriés.

Les boucles : for, while, do-while

Les boucles permettent de répéter un bloc de code plusieurs fois. PHP propose trois types de boucles :

La boucle for :

  • Utilisée lorsqu'on connaît à l'avance le nombre d'itérations.
  • for ($i = 0; $i < 10; $i++)

La boucle while :

  • Utilisée lorsque l'on répète une action tant qu'une condition est vraie.
  • while ($i < 10)

La boucle do-while :

  • La condition est testée après l'exécution du bloc de code. Cela garantit que le bloc s'exécute au moins une fois.
  • do { ... } while ($i < 10);

Exercice :

  • Créez une boucle for qui affiche les nombres de 1 à 10, une boucle while pour afficher les nombres de 10 à 1, et une boucle do-while qui affiche les nombres de 1 à 5.

Les switch / case

La structure switch est utilisée pour exécuter un bloc de code en fonction de la valeur d'une variable. Elle est souvent plus pratique que plusieurs if imbriqués lorsqu'il y a plusieurs valeurs à tester :

  • switch : Permet de tester une variable contre plusieurs valeurs possibles.
  • case : Chaque valeur testée, si elle correspond à la variable, exécute le bloc associé.
  • default : Exécute le bloc si aucune des valeurs case ne correspond.

Exercice :

  • Créez un switch pour afficher les jours de la semaine en fonction d'un nombre (1 = Lundi, 2 = Mardi, etc.).
