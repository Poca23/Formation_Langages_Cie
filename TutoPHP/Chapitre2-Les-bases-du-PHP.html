Chapitre 2 - Les bases du PHP

Les variables et les types de données

En PHP, les variables sont utilisées pour stocker des valeurs. Vous n'avez pas besoin de déclarer le type de données de la variable, PHP le fait automatiquement. Voici les types de données principaux :

  • Entiers : Des nombres sans décimales, comme 25.
  • Flottants : Des nombres avec des décimales, comme 3.14.
  • Booléens : Soit true ou false.
  • Chaînes de caractères : Texte entre guillemets, comme "Bonjour".
  • Tableaux : Des collections de valeurs, comme $nombres = [1, 2, 3];.

Exercice :

  • Déclarez une variable pour stocker votre âge et affichez-la avec la fonction echo.

Opérateurs arithmétiques et logiques

PHP fournit plusieurs opérateurs pour effectuer des calculs (arithmétiques) ou des comparaisons (logiques) :

Opérateurs arithmétiques :

  • + Addition
  • - Soustraction
  • * Multiplication
  • / Division
  • % Modulo (reste de la division)

Opérateurs logiques :

  • == égal à
  • === strictement égal (même type et valeur)
  • != différent de
  • && ET logique
  • || OU logique

Exercice :

  • Créez deux variables pour stocker des nombres et effectuez une addition et une comparaison entre eux.

Les fonctions de base (echo, print, var_dump)

Les fonctions suivantes sont utilisées pour afficher des données :

  • echo : Affiche du texte ou des variables.
  • print : Similaire à echo, mais retourne une valeur (toujours 1).
  • var_dump : Affiche des informations détaillées sur une variable (type et valeur).

Exercice :

  • Créez une variable et affichez-la avec echo, print, et var_dump.

Introduction aux constantes et aux variables globales

Les constantes sont des valeurs qui ne changent jamais pendant l'exécution du programme. Elles sont définies avec la fonction define().

  • Les constantes : Par exemple, define("PI", 3.14);
  • Les variables globales : Vous pouvez accéder à une variable globale depuis n'importe où, en utilisant le mot-clé global dans une fonction.

Exercice :

  • Créez une constante pour stocker la valeur de Pi et une variable globale. Affichez-les dans une fonction.
