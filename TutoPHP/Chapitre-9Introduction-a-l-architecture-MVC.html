Chapitre 9 - Introduction à l'architecture MVC (Model-View-Controller)

Qu'est-ce que l'architecture MVC ?

L'architecture MVC est un modèle de conception qui sépare une application en trois parties distinctes : le modèle (Model), la vue (View) et le contrôleur (Controller).

  • Model : Représente les données et la logique de l'application.
  • View : Gère l'affichage des données à l'utilisateur (interface utilisateur).
  • Controller : Traite les entrées de l'utilisateur, interagit avec le modèle et met à jour la vue.

Le but de l'architecture MVC est de séparer les différentes préoccupations dans une application, ce qui facilite la maintenance et les évolutions de celle-ci.

Création d'une petite application PHP MVC

Nous allons maintenant créer une petite application utilisant l'architecture MVC en PHP.

1. Le Model (Modèle) :

Le modèle est responsable de la gestion des données. Dans notre exemple, nous allons créer un modèle simple pour gérer des tâches à faire.

class TaskModel {
    private $tasks = [];

    public function addTask($task) {
        $this->tasks[] = $task;
    }

    public function getTasks() {
        return $this->tasks;
    }
}

2. La View (Vue) :

La vue est responsable de l'affichage des données. Elle peut être un fichier HTML simple qui affiche les tâches.

<!-- view.php -->
<?php foreach ($tasks as $task) : ?>
    <p><?php echo $task; ?></p>
<?php endforeach; ?>

3. Le Controller (Contrôleur) :

Le contrôleur interagit avec le modèle et met à jour la vue. Il prend les actions de l'utilisateur, comme ajouter une tâche, et met à jour la vue en conséquence.

class TaskController {
    private $model;
    private $view;

    public function __construct($model, $view) {
        $this->model = $model;
        $this->view = $view;
    }

    public function addTask($task) {
        $this->model->addTask($task);
        $this->view->render($this->model->getTasks());
    }
}

4. Intégration des trois parties :

Enfin, nous intégrons le modèle, la vue et le contrôleur pour faire fonctionner notre petite application MVC.

$model = new TaskModel();
$view = new TaskView();
$controller = new TaskController($model, $view);

$controller->addTask("Apprendre PHP");
$controller->addTask("Pratiquer Laravel");

Le contrôleur prend en charge l'ajout de nouvelles tâches et met à jour la vue avec la liste des tâches.

  • Créez une vue pour afficher les contacts et un formulaire pour en ajouter.
