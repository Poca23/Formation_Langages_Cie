Chapitre 5 - Connexion à une base de données

Retour au Sommaire

Introduction

Dans ce chapitre, nous allons apprendre comment connecter une application Java à une base de données MySQL. Une base de données est essentielle pour stocker, récupérer et manipuler des données dans une application backend.

Étape 1 : Installation de MySQL

Avant de pouvoir établir une connexion, vous devez avoir MySQL installé sur votre machine. Vous pouvez télécharger MySQL à partir de son site officiel et suivre les instructions d'installation pour votre système d'exploitation.

Vérifiez que MySQL fonctionne en exécutant la commande suivante dans votre terminal :

mysql --version

Étape 2 : Ajouter la dépendance MySQL dans votre projet Java

Pour connecter Java à une base de données MySQL, vous devez utiliser un driver JDBC. Le driver JDBC de MySQL est disponible via Maven. Ajoutez la dépendance suivante dans votre fichier pom.xml : 


<dependency>
    <groupId>mysql</groupId>
    <artifactId>mysql-connector-java</artifactId>
    <version>8.0.28</version>
</dependency>

Une fois la dépendance ajoutée, Maven téléchargera automatiquement le driver JDBC de MySQL.

Étape 3 : Configurer la connexion à MySQL

Pour vous connecter à la base de données MySQL, vous devez spécifier l'URL de connexion, le nom d'utilisateur et le mot de passe. Voici un exemple de code Java pour établir la connexion :


import java.sql.Connection;
import java.sql.DriverManager;
import java.sql.SQLException;

public class DatabaseConnection {
    public static void main(String[] args) {
        String url = "jdbc:mysql://localhost:3306/nom_de_la_base";
        String user = "root";
        String password = "votre_mot_de_passe";

        try {
            Connection connection = DriverManager.getConnection(url, user, password);
            System.out.println("Connexion réussie !");
        } catch (SQLException e) {
            System.out.println("Erreur de connexion : " + e.getMessage());
        }
    }
}

Assurez-vous de remplacer nom_de_la_base par le nom réel de votre base de données MySQL et d'utiliser le bon mot de passe pour l'utilisateur MySQL.

Étape 4 : Tester la connexion

Une fois que vous avez configuré la connexion, exécutez votre programme Java. Si la connexion est réussie, le message "Connexion réussie !" devrait s'afficher. En cas d'erreur, vérifiez les informations de connexion et assurez-vous que le serveur MySQL est en cours d'exécution.

Conclusion

Dans ce chapitre, nous avons appris à connecter une application Java à une base de données MySQL. Cette connexion est la base pour interagir avec les données dans votre application backend. Vous pouvez maintenant effectuer des opérations CRUD (Créer, Lire, Mettre à jour, Supprimer) sur la base de données en utilisant Java et MySQL.
