Chapitre 2 - Installation et Configuration de l'Environnement

Retour au Sommaire

1. Installation de Java Development Kit (JDK)

Le JDK (Java Development Kit) est indispensable pour compiler et exécuter vos programmes Java. Voici les étapes pour l'installer :

  1. Rendez-vous sur le site officiel d'Oracle ou utilisez une version open source comme OpenJDK.
  2. Téléchargez la version correspondant à votre système d'exploitation (Windows, macOS, Linux).
  3. Suivez les instructions d'installation spécifiques à votre plateforme.
  4. Une fois installé, vérifiez que Java est bien configuré en exécutant la commande suivante dans votre terminal : 
    java -version

2. Installation d'un IDE (Environnement de Développement Intégré)

Un IDE facilite l'écriture et la gestion de projets Java. Nous recommandons les options suivantes :

  • IntelliJ IDEA : Puissant et populaire, avec une version Community gratuite. Télécharger IntelliJ IDEA
  • Eclipse : Un choix classique pour les développeurs Java. Télécharger Eclipse
  • VS Code : Léger, avec l'extension Java Extension Pack pour Java. Télécharger VS Code

Installez l'IDE de votre choix et configurez-le pour votre premier projet Java.

3. Configuration des Variables d'Environnement

Pour que votre système reconnaisse les commandes Java, ajoutez le chemin du JDK à votre variable d'environnement PATH :

  1. Localisez le dossier d'installation de votre JDK (par exemple, C:\\Program Files\\Java\\jdk-version sur Windows).
  2. Ajoutez le chemin bin à votre variable PATH.
  3. Redémarrez votre terminal et vérifiez avec : 
    echo %PATH% (Windows) ou echo $PATH (Linux/macOS).

4. Création du Premier Projet Java

Voici les étapes pour créer et exécuter un premier programme Java :

  1. Ouvrez votre IDE et créez un nouveau projet Java.
  2. Ajoutez une nouvelle classe avec un fichier nommé Main.java.
  3. Écrivez le code suivant dans le fichier : 
    public class Main {
    public static void main(String[] args) {
        System.out.println("Bonjour, Java Backend !");
    }
}
  4. Exécutez le programme et observez le message s'afficher dans la console.
