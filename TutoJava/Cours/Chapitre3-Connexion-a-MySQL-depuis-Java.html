Chapitre 3 - Connexion à MySQL depuis Java

Retour au Sommaire

1. Présentation de JDBC

JDBC (Java Database Connectivity) est une API Java permettant de se connecter à une base de données relationnelle et de manipuler les données. JDBC agit comme un pont entre les applications Java et les bases de données, telles que MySQL, PostgreSQL ou Oracle.

2. Installation du Connecteur MySQL

Pour utiliser MySQL avec JDBC, vous devez télécharger et ajouter le connecteur MySQL à votre projet Java :

  1. Téléchargez le connecteur depuis le site officiel : Télécharger MySQL Connector/J.
  2. Ajoutez le fichier JAR du connecteur à votre projet :
    • Avec IntelliJ IDEA : Faites un clic droit sur votre projet, choisissez "Add Library", et sélectionnez le fichier JAR.
    • Avec Eclipse : Faites un clic droit sur votre projet, choisissez "Build Path > Add External Archives", et ajoutez le fichier JAR.

3. Établir une Connexion à la Base de Données

Voici un exemple de code Java pour se connecter à une base de données MySQL : 

import java.sql.Connection;
import java.sql.DriverManager;
import java.sql.SQLException;

public class Main {
    public static void main(String[] args) {
        String url = "jdbc:mysql://localhost:3306/nom_de_la_base";
        String user = "nom_utilisateur";
        String password = "mot_de_passe";

        try (Connection connection = DriverManager.getConnection(url, user, password)) {
            System.out.println("Connexion réussie à la base de données !");
        } catch (SQLException e) {
            System.out.println("Erreur de connexion : " + e.getMessage());
        }
    }
}

Assurez-vous de remplacer nom_de_la_base, nom_utilisateur et mot_de_passe par vos propres informations.

4. Tester la Connexion

Pour tester le code ci-dessus :

  1. Assurez-vous que votre serveur MySQL est démarré.
  2. Exécutez le programme Java dans votre IDE.
  3. Si la connexion est réussie, le message suivant s'affichera dans la console : "Connexion réussie à la base de données !"
  4. En cas d'erreur, vérifiez les informations de connexion et les logs.
