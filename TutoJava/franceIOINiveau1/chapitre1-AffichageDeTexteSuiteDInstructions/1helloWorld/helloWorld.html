Afficher du texte :

En Java, le programme minimal (qui ne fait rien) est :


    class Main {
        public static void main(String[] args) {
        }
    }

Nous allons compléter ce programme minimal pour qu'il fasse quelque chose. Explications…

Pour afficher du texte, on écrit System.out.println puis, entre parenthèses, le texte à afficher, entre guillemets.
Ainsi, pour afficher le mot « Bonjour », on écrira le code suivant : 


    class Main {
        public static void main(String[] args) {
            System.out.println("Bonjour");
        }
    }

En effet, println vient de l'anglais print line qui signifie « imprimer une ligne ».

Chaque ligne du programme terminée par un point-virgule ; représente une instruction.
