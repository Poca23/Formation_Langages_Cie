Étapes Git Interactives

1. Initialisation

# Initialiser le projet Git
git init

# Nouvellement, forcer la branche principale en "master"
git branch -M master

2. Ajout du dépôt distant

# Ajouter le dépôt distant (par exemple GitHub)
git remote add origin https://github.com//.git

# Vérifier que le remote a été ajouté correctement
git remote -v

3. Création de branches

# Créer et organiser les branches
git checkout -b dev
git checkout -b front
git checkout dev
git checkout -b back

4. Commits initiaux

# Ajouter tous les fichiers dans "back"
git checkout back
git add .
git commit -m "Initialisation du Back-end" --date="2023-11-01T12:00:00"

# Ajouter tous les fichiers dans "front"
git checkout front
git add .
git commit -m "Initialisation du Front-end" --date="2023-11-01T11:00:00"

5. Fusion des Branches

# Fusion des branches dans "dev"
git checkout dev
git merge front
git merge back

# Fusion finale dans "master"
git checkout master
git merge dev

6. Pousser les branches vers le dépôt distant

# Pousser la branche master
git push origin master

# Pousser les branches secondaires (back, front, dev)
git push origin dev
git push origin front
git push origin back

# Si jamais le "remote tracking" n'est pas encore configuré :
git push -u origin

7. Vérifications et historique

# Vérifier toutes les branches
git branch -a

# Examiner l'historique des commits
git log --oneline --graph --all
