Variables et etypes de donnees

variable : un espace mémoire ou une valeur peut petre stockée.

type de données :

structures de donnees

tableaux ou liste (array) : plsr valeurs indexées

par defaut indexes des tableaux sont des nombres de type int et commence toujours par 0.

exemple :
